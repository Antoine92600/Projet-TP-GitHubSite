# Projet TP GitHub

## Description
Ce projet contient des fichiers `.alg` destinés à être utilisés avec **AlgoBox**. L'objectif du projet est de créer et tester des algorithmes dans l'environnement AlgoBox. Vous pouvez ouvrir ces fichiers et exécuter les algorithmes directement dans AlgoBox.

## Prérequis
Avant de pouvoir utiliser ce projet, vous devez avoir les logiciels suivants installés :
- **Git** : pour cloner ce projet depuis GitHub (téléchargeable sur https://git-scm.com).
- **AlgoBox** : pour ouvrir et tester les fichiers `.algo` (téléchargeable sur https://www.algo-box.fr/).

## Instructions d'exécution

### Cloner le projet
Pour cloner ce projet, vous devez d'abord avoir **Git** installé sur votre machine. Une fois Git installé, ouvrez un terminal (PowerShell ou Git Bash) et exécutez la commande suivante pour cloner ce dépôt :

```bash
git clone https://github.com/Antoine92600/Projet-TP-GitHubSite.git
